# Estudo de Limite de Segurança em Vendas 📈

Este projeto consiste em um script Python simples desenvolvido para analisar o desempenho de vendas em relação a um teto de segurança pré-definido. Ele calcula a média de um período de vendas e valida se os resultados estão dentro dos parâmetros operacionais seguros.

## 🚀 Funcionalidades

O script realiza as seguintes etapas:
1.  **Definição de Parâmetro**: O usuário define um valor inteiro como `limite_segurança`.
2.  **Entrada de Dados**: Coleta o valor de três vendas distintas.
3.  **Processamento**:
    * Soma o valor total das vendas.
    * Calcula a média aritmética dos resultados.
4.  **Análise Crítica**: Compara a média obtida com o limite de segurança inicial.
5.  **Feedback**: Informa ao usuário se o valor é "Seguro" ou "Não seguro".

## 🛠️ Tecnologias Utilizadas

* **Python 3**: Linguagem base do projeto.
* **Google Colab**: Ambiente onde o script foi originalmente gerado.
